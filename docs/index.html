VoiceShift

high-quality real-time voice conversion system

Installation

Install Python 3.6 and Pipenv on your machine then download the voice conversion system on Github. The project has dependencies that need to be installed.

  1. Run pipenv install on root directory of the project.
  2. Enter the source directory of the project with cd source.
  3. Use python cli.py --help to show command-line interface details.

Usage

Currently, the system only supports WAV format. Hyperparameters and directories can be set inside the .env file found on source directory and the sampling rate inside must also match the sampling rate of the audio files. The next updates would address these issues and might also provide a friendly graphical user interface.
